# Roadmap — School Portal

> Идеи и планы по развитию проекта. Порядок — от самых реалистичных к самым амбициозным.

---

## 1. Онлайн-уроки (WebRTC) 🎥
Возобновить разработку онлайн-уроков: учитель показывает экран, ученики видят, чат, микрофон. В этой версии уже заложена основа (комнаты, WebSocket), но она не была доведена до ума.

## 2. Автоперевод учеников в следующий класс 🔄
Каждый год в конце августа — скрипт, который переводит 5А → 6А, 6Б → 7Б и т.д., а 11-е классы выпускает. Директор нажимает кнопку — всё делается само.

## 3. Электронное расписание звонков 🔔
Виджет на панели учителя/ученика: показывает текущий урок, сколько осталось до перемены, какой урок следующий.

## 4. Родительский кабинет 👨‍👩‍👧‍👦
Отдельная роль «родитель»: видит оценки только своих детей, домашние задания, может общаться с учителями через чат. Привязка через email или QR-код.

## 5. Push-уведомления в Telegram / VK 📱
Интеграция с Telegram Bot API: новая оценка → сразу в Telegram родителю, новое домашнее задание → в чат класса, напоминание о родительском собрании.

## 6. Умный экспорт в Excel / CSV 📊
Выгрузка ведомости за период: оценки всего класса по предмету, средний балл, количество пропусков — одним кликом.

## 7. Генератор характеристик на ученика 📝
Шаблоны с подстановкой: ФИО, класс, средний балл, поведение. Директор выбирает ученика — характеристика готова.

## 8. Расписание замен (подмены) 🆘
Если учитель заболел — администратор ставит замену, система уведомляет нового учителя и класс.

## 9. Модуль питания 🍽
Меню на неделю, учёт оплаты обедов, отметка кто ел / не ел. Для бухгалтерии — отчёт по питанию за месяц.

## 10. Электронная зачётка / портфолио ученика 🏆
Раздел «Мои достижения»: грамоты, сертификаты, результаты олимпиад. Можно прикреплять скан/фото. Потом распечатать красивое портфолио.

## 11. API для сторонних приложений 🔌
REST API с токеном для мобильного приложения или интеграции с 1С / бухгалтерией.

## 12. Модуль тестирования / контрольных работ ✍️
Учитель создаёт тест (выбор варианта, открытый вопрос), ученики проходят онлайн, система сама проверяет и ставит оценку.

## 13. Календарь событий школы 📅
Общий календарь: каникулы, собрания, праздники, олимпиады, спортивные мероприятия. Экспорт в .ics (совместимость с Google Calendar / Apple Calendar).

## 14. Посещаемость по QR-коду 📱
Учитель генерирует QR на уроке, ученик сканирует — отметка о присутствии. Исключает «забыл отметить» и лишнюю бумажную работу.

## 15. Многоязычность (i18n) 🌍
Добавить английский, казахский, узбекский и другие языки. Сделать проект доступным не только для русскоязычных школ.
